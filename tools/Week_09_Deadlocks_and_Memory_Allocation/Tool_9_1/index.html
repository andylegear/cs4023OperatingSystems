Resource Allocation Graph (RAG) — Process nodes (circles) request resource nodes (rectangles). An assignment edge goes from resource → process (resource is held). A request edge goes from process → resource (process wants it). Cycle rule: if all resources have a single instance, a cycle means deadlock. For multi-instance resources, a cycle is necessary but not sufficient — use Banker's Algorithm.

Banker's Algorithm (Dijkstra, 1965) — Given the current Allocation, Max demand, and Available resources, determine whether the system is in a safe state: a state where a safe sequence exists so that every process can eventually complete. At each step, find any process whose Need ≤ Work; run it, release its resources, repeat until all processes finish (SAFE) or none can proceed (UNSAFE).
