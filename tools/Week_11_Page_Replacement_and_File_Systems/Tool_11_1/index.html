What is Belady's Anomaly?

On the reference string 1, 2, 3, 4, 1, 2, 5, 1, 2, 3, 4, 5:

  • FIFO with 3 frames: 9 faults
  • FIFO with 4 frames: 10 faults ← MORE frames, MORE faults!

FIFO is not a stack algorithm — its set of pages with n+1 frames is not always a superset of pages with n frames.

LRU and OPT are stack algorithms — they are immune to Belady's anomaly. More frames always means ≤ page faults.
