# CS4023 Operating Systems — Public Resources

University of Limerick · Academic Year 2025–26

This repository hosts publicly shared teaching resources for students enrolled in **CS4023 Operating Systems**.

## What's here

| Folder | Contents |
|---|---|
| `slides/` | Interactive Slidev HTML presentations (open in browser) |
| `lectures/` | Downloadable PPTX lecture slides |
| `tools/` | Active learning tools — drag-and-drop exercises, no login required |
| `tutorials/` | Tutorial question sheets (Word/docx) |
| `labs/` | Lab instruction sheets (Word/docx) |
| `index.html` | Root navigation page linking all resources by week |

## How to use

- Browse the [index page](./index.html) for a week-by-week overview
- **Live slides** open directly in your browser — use arrow keys or swipe to navigate
- **PPTX and docx files** download automatically — open with Microsoft Office or Google Docs
- **Active learning tools** work on mobile and desktop, no login needed

## Module overview

CS4023 covers the core principles of modern operating systems across 13 teaching weeks:

1. Introduction & OS Overview
2. Computer Organisation & OS Structure
3. Processes
4. Threads
5. Interprocess Communication
6. CPU Scheduling
7. Synchronisation (+ Midterm)
8. Concurrency & Monitors
9. Deadlocks & Memory Allocation
10. Paging & Virtual Memory
11. Page Replacement & File Systems
12. I/O Systems & Virtualisation
13. Cloud, AI & Operating Systems

## Textbook

Silberschatz, Galvin & Gagne — *Operating System Concepts*, 10th ed.
Available as an e-book through the UL Library.
